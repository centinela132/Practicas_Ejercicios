Repositorio con ejercicios y prácticas realizadas del curso **Flutter-Dart**.

## ¿Qué es Flutter?
Flutter es un **framework de desarrollo** creado por **Google** para construir aplicaciones móviles, web y de escritorio de manera rápida y eficiente, usando una única base de código. Permite a los desarrolladores crear interfaces de usuario (UI) nativas, de alto rendimiento, con una experiencia visual atractiva y consistente en diferentes plataformas, como **iOS**, **Android**, **macOS**, **Windows** y la **web**.

## ¿Qué es Dart?
**Dart** es el **yenguaje de programación** utilizado para desarrollar aplicaciones con **Flutter**. Desarrollado también por **Google**, Dart es fácil de aprender y optimizado para crear aplicaciones rápidas y reactivas. Es un lenguaje orientado a objetos, con una sintaxis similar a otros lenguajes populares como **JavaScript** y **Java**.

## ¿Por qué usar Flutter y Dart?

- **Desarrollo multiplataforma**: Permite escribir una sola base de código que funciona en **iOS**, **Android**, **Web**, y otras plataformas.
  
- **Rendimiento nativo**: Las aplicaciones creadas con Flutter se compilan directamente a **código nativo**, lo que resulta en un rendimiento comparable al de las aplicaciones nativas.

- **UI personalizable**: Flutter incluye una amplia gama de **widgets** personalizables que hacen posible crear interfaces visualmente atractivas y únicas.

- **Productividad**: Con **hot reload**, los desarrolladores pueden ver los cambios en tiempo real sin reiniciar la app.

## ¿A quién va dirigido?
Flutter está dirigido a **desarrolladores móviles**, **web** y **de escritorio**, desde principiantes hasta avanzados, que buscan crear aplicaciones multiplataforma con alto rendimiento y buen diseño.
